# Official Prajwalan '19 Mobile App

https://play.google.com/store/apps/details?id=com.nadeem.prajwalan

**Prerequisites:**
1. npm
2. ionic, cordova

**Steps (in console):**
1. npm install
2. ionic serve (in browser- auto reload)

**For android apk:**
1. ionic cordova platform add android
2. ionic cordova build android
